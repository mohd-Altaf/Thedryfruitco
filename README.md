# The Dry Fruit Co

Welcome to the official repository for **The Dry Fruit Co** website.

🌐 **Live Website:** https://mohd-altaf.github.io/thedryfruitco/

## About

The Dry Fruit Co is a product-focused website created to showcase premium dry fruits, healthy snacks, and natural food products through a clean and responsive online presence.

This repository contains the complete source code for the website.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* `index.html` — Main landing page
* `style.css` — Website styling and layout
* `script.js` — Interactive functionality
* `images/` — Product images and media assets

## Features

* Clean and modern UI
* Responsive design for mobile and desktop
* Product showcase layout
* Fast-loading static website
* Easy to maintain codebase
